Here is the description of the tables :

contents :

| Field            | Type           | Description                                                      |
|------------------|----------------|------------------------------------------------------------------|
| id               | SERIAL         | Primary key                                                      |
| title            | VARCHAR(255)   | Content title                                                    |
| content_type     | VARCHAR(20)    | "poem", "reflection", "drawing" (ENUM or FK to `types`)          |
| content_text     | TEXT           | NULL if content_type = "drawing" (text is on the image)          |
| published_at     | TIMESTAMP      | Publication date (auto-filled)                                   |
| image_id         | INTEGER        | Foreign key to images (NULL if no image)                         |
| video_id         | INTEGER        | Foreign key to videos (NULL if no video)                         |

images :

| Field   | Type         | Description                       |
|---------|--------------|-----------------------------------|
| id      | SERIAL       | Primary key                      |
| url     | VARCHAR(512) | URL or path to the image        |
| caption | VARCHAR(255) | Caption or description            |
| is_ai   | BOOLEAN      | True if generated by AI            |

videos :

| Field | Type         | Description                                   |
|-------|--------------|-----------------------------------------------|
| id    | SERIAL       | Primary key                                  |
| url   | VARCHAR(512) | YouTube URL (e.g., https://youtu.be/xxxx)       |
| title | VARCHAR(255) | Video title                             |

themes :

| Field | Type         | Description                         |
|-------|--------------|-------------------------------------|
| id    | SERIAL       | Primary key                        |
| name  | VARCHAR(100) | Theme name (e.g., "Love", "Nature") |

content_themes :

| Field       | Type     | Description                   |
|-------------|----------|-------------------------------|
| content_id  | INTEGER  | Foreign key to contents   |
| theme_id    | INTEGER  | Foreign key to themes     |

tags :

| Field | Type         | Description                             |
|-------|--------------|-----------------------------------------|
| id    | SERIAL       | Primary key                            |
| name  | VARCHAR(100) | Tag name (e.g., "Melancholy", "Spring") |

content_tags :

| Field       | Type     | Description                 |
|-------------|----------|-----------------------------|
| content_id  | INTEGER  | Foreign key to contents |
| tag_id      | INTEGER  | Foreign key to tags     |

links :

| Field            | Type     | Description                                        |
|------------------|----------|----------------------------------------------------|
| id               | SERIAL   | Primary key                                       |
| content_id       | INTEGER  | Foreign key to contents (source content)       |
| target_content_id | INTEGER  | Foreign key to contents (recommended content)   |
